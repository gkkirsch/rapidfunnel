RapidFunnel Universal Tracker Script

This page hosts the minified JavaScript file for the RapidFunnel Universal Tracker.

The script itself can be included in your website using:

<script src="https://gkkirsch.github.io/rapidfunnel/rapidfunnel-tracker.min.js" defer></script>
